I only use [Codeberg](https://codberg.org) for publishing source code for my projects and my profile is available [here](https://codeberg.org/eyekay/).

Despite my reluctance to have a account on GitHub, in today's world it is simply impossible to be a FOSS contributor and not have an GitHub account, as the vast majority of open source development happens here. It is extremely ironic, therefore, that GitHub itself is proprietary software, and the platform is run by a multi-billion corporation which is openly hostile to adoption of open source source software and standards.

Do not use GitHub for hosting your projects - use a free and open source and community-run alternative!
